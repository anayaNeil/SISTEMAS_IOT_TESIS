Descripción de Sistema Electrico IOT

  • Interruptor On/Off

    • El interruptor se ubica en la parte izquierda del equipo y permite encendedido o apagado de este .

  • 1. Indicador Led de advertencia

    • El indicador Led de advertencia indica que la temperatura esta fuera de rango.

  • 2. Pantalla Oled

    • La pantalla Oled muestra la temperatura y la olla seleccionada actualmente.

  • 3. Boton selector de OLLAS

    • El boton selector de OLLAS permite seleccionar la olla con la que se desea trabajar.

  • 4. Sensor de Temperatura

    • El sensor de temperatura mide la temperatura del agua en la olla.

  • 5. Alimentación

    • La alimentación es de 5 Voltios dc y proporciona la energía necesaria para el funcionamiento del sistema Electrico.

  • Bocina de Alarma

    • La bocina de alarma produce un sonido sonoro cuando se produce una advertencia.

Etapas en proceso de cerveza

    El sistema consta de tres ollas escalonadas diseño para procesar cerveza en diferentes etapas del proceso, cada una trabajando a una temperatura específica o dentro de un rango. Las ollas está conectadas mediante válvulas que permiten transferir el contenido entre ellas de manera controlada.

    Pasos Importantes a Seguir

    • 1. Preparación del Sensor de Temperatura
      • Antes de iniciar, desinfecte el sensor de temperatura.
      • Inserte suavemente el sensor en su puerto de entrada .
    • 2. Cambio entre Ollas
      • Al cambiar a una olla diferente, presione el botón selector en el equipo para indicar el cambio.
      • Una vez seleccionado, no manipule el botón hasta que sea necesario cambiar nuevamente de olla.
    • 3. Finalización del Proceso
      • Al terminar, apague el equipo utilizando la palanca lateral izquierda (ON/OFF).
      • Retire el sensor de temperatura y desinfecte nuevamente la parte metálica antes de guardarlo.

OLLA 1

Precalentado

  • Estado inicial: Por defecto, el sistema está configurado en "OLLA 1".

Preparación:

  • Introducir el sensor de temperatura (parte metálica sumergida a unos 15 cm).
  • Calentar el agua hasta alcanzar 75°C.

Propósito:

  • Este paso prepara el agua para que, al mezclarse en la Olla 2 con la malta, la temperatura baje a un promedio de 67°C.

Acciones automáticas:

  • Al alcanzar los 75°C, el sistema:
    • Emite una alerta sonora durante 6 segundos.
    • La bocina dejara de sonar pero el LED rojo como indicador seguira encendedido indicando que ya esta en el punto adecuado y debe pasar a la siguiente etapa.

OLLA 2

Proceso de Macerado

  • Estado inicial: Seleccionar "OLLA 2" en el sistema presionando el botón selector.

Preparación:

  • Retirar el sensor de temperatura de la Olla 1 y colocarlo en la Olla 2 a la misma profundidad.
  • Agregar la malta al agua siguiendo la proporción 3:1 agua/malta.

Proceso:

  • Mantener la temperatura entre 65°C y 70°C durante 60 minutos.
  • Si la temperatura se desvía del rango (supera los 70°C o cae por debajo de los 65°C), se debe:
    • Ajustar mediante calentamiento (prender el fuego) o enfriamiento (por agitación).

Acciones automáticas:

  • Notificación de alerta sonora y activación del LED indicador ROJO cuando la temperatura se sale del rango establecido (70°C y 65°C) durante el proceso, la bocina dejara de sonar pero el LED rojo seguira encendedido indicando que sigue en un punto critico.

OLLA 3

Ebullición y Enfriado

  • Estado inicial: Seleccionar "OLLA 3" en el sistema presionando el botón selector.

Preparación:

  • Retirar el sensor de temperatura de la Olla 2 y colocarlo en la Olla 3 a la misma profundidad.

Proceso:

  • Ebullición:
    • Elevar la temperatura hasta el punto de ebullición y mantenerla durante 60 minutos.
  • Enfriado:
    • Una vez se cumplio con el tiempo de ebullición, se debe Reducir la temperatura rápidamente hasta 30°C.

Propósito:

  • Preparar el mosto para la adición de levaduras y prevenir contaminación.

Acciones automáticas:

  • Notificación de alerta cuando la temperatura descienda hasta los 30°C.
  • Acción manual: Agregar las levaduras para continuar con el proceso de fermentación.

Descargar Historial

Características de la función

  • La función Descargar Historial permite obtener un registro completo de las temperaturas recopiladas a lo largo de todo el proceso de elaboración de la cerveza.

Procedimiento de descarga:

  • Presionar el boton Mostrar Historial para obetener los datos a los largo de todo el proceso el proceso.
  • Ubicar el botón de tres barras localizado en la esquina superior derecha del gráfico de temperatura.
  • Al hacer clic en este botón, se despliega un menú con las opciones de exportación disponibles.
  • Seleccionar el formato deseado (imagen o datos).
  • La descarga se iniciará automáticamente.

Formatos de descarga:

  • Gráficos:
    • PNG, JPEG, SVG.
  • Datos:
    • CSV, XLS.
